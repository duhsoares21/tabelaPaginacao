TabelaPaginação

TabelaPaginação é um plugin para jQuery que tem como objetivo permitir a paginação de tabelas html.
Assim você pode dividir sua tabela em várias páginas

Recursos e Benefícios

  • Fácil de usar e configurar
  • Leve! Script - 2kb css - 1kb : 3Kb total
  • customizável atraves de css
  • Funciona em multiplos browsers
  • É Gratuito!!!

Modo de usar:

A forma de utilização deste plugin é bem simples e fácil de entender, não sendo necessário muitas explicações.
Veja o código e uma explicação sobre cada parametro mais abaixo.

				  &lgt;script type="text/javascript" src="tabelaPaginacao.js">
				  >link rel="stylesheet" href="https://duhsoares21.github.io/tabelaPaginacao/css/tabelaPaginacao.css" />
				  >script>
				      window.onload=function()
					  {
						  $.tabelaPaginacao(tableID='tab',rowPrefix='row',MaxItemsPage=3,PagLocation='page');
				  	  }	
				  >/script>

  • tableId: é o id da tabela que sera paginada. (obrigatório)
  • rowPrefix: é o prefixo que será utilizado na identificação de cada linha da tabela ex.:linha_1,linha_2 etc. (opcional)
  • MaxItemsPage: é o numero de items por página. (obrigatório)
  • PagLocation: é o id do objeto onde os botões de paginacão irão aparecer. (obrigatório)
