TabelaPaginação

TabelaPaginação is a jQuery plugin that allow you to paginate html tables.
This way you can split your table in many pages

Features and Advantages

  • Easy to use and setup
  • lightweight! Script - 2kb css - 1kb : 3Kb total
  • customizable using css
  • It's cross-browser
  • It's Free!!!

Mode of use:

The form of use it's easy and simple to understand and the code explain it self.
See the code and a explanation of the parameters bellow.

				  <script type="text/javascript" src="tabelaPaginacao.js">
				  <link rel="stylesheet" href="tabelaPaginacao.css" />
				  <script>
				      window.onload=function()
					  {
						  $.tabelaPaginacao(tableID='tab',rowPrefix='row',MaxItemsPage=3,PagLocation='page');
				  	  }	
				  </script>

  • tableId: It's the Id of the table who be paginated. (required)
  • rowPrefix: It's the prefix that will be used on the identification of every table row ex.:row_1,row_2 etc. (optional)
  • MaxItemsPage: It's the number of items per page. (required)
  • PagLocation: It's the id of the object where the paginate buttons will be appended. (required)

Compatibility:

The following browsers are supported: Internet Explorer 6+, Mozilla Firefox, Google Chrome, Safari e Opera
